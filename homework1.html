特雷西.麦格雷迪

特雷西·麦格雷迪（Tracy McGrady），1979年5月24日出生于美国佛罗里达州巴托，前美国职业篮球运动员，司职得分后卫/小前锋，绰号“T-Mac”、“麦迪”/“麦蒂”。

在1997年NBA选秀大会中，麦迪以高中生球员的身份，在第1轮第9顺位被多伦多猛龙队选中。职业生涯曾先后效力于多伦多猛龙队、奥兰多魔术队、休斯敦火箭队、纽约尼克斯队、底特律活塞队、亚特兰大老鹰队、CBA青岛双星队及圣安东尼奥马刺队。在16年的职业生涯中，麦迪2次荣膺NBA得分王，7次入选NBA全明星阵容，2次入选NBA最佳阵容一阵，3次入选NBA最佳阵容二阵，2次入选NBA最佳阵容三阵，2001年荣获NBA进步最快球员奖。

2004年12月9日，麦迪在比赛的最后35秒中狂砍13分，并命中绝杀战胜对手，成就经典的“麦迪时刻” [1] 。2013年8月26日，麦迪正式宣布退役 [1] 。2017年4月2日，麦迪正式入选2017年奈·史密斯篮球名人纪念堂 [2] 。

早年经历

童年时期

1979年5月24日，特雷西·麦格雷迪出生于美国佛罗里达州巴托童年时期的麦迪童年时期的麦迪(3张)市 [3] 。麦格雷迪很小就表现出了运动天赋。他最先迷上的并不是篮球，而是棒球。少年麦格雷迪长得高瘦结实，他是一名快速有力的击球手，跑垒也十分强劲。后来，麦格雷迪开始和朋友们一起打篮球 [4] 。1993年，安芬尼·哈达威被金州勇士队以“探花秀”的身份选中，麦格雷迪开始以安芬尼·哈达威为目标，效仿他在比赛中的招式 [5] 。

高中时期

1994年，麦格雷迪就读于奥本戴尔高中。1997年高中三年级时，麦格雷迪场均可以得到23分和12个篮板 [5] 。由于奥本戴尔高中对麦格雷迪的治学态度持疑，麦格雷迪决定转入蒙特赛昂基督学校。麦格雷迪率领学校一路攀升到全美实力排名第二位，并且被《今日美国》评为“年度最佳高中球员”，同时还被美联社评为“北卡莱罗纳州年度最佳球员”。1996-97赛季，麦格雷迪场均得到27.5分8.7个篮板2.8抢断，赛季结束后麦格雷迪决定跳过大学，以高中生球员的身份直接参加NBA选秀 [5] 。

运动生涯

NBA经历

猛龙时期

1997年6月25日，麦格雷迪以高中生球员的身份，在1997年NBA选秀大会第1轮第9顺位被多伦多猛龙队选中 [6] 。

1997年11月15日，在猛龙77-105负于印第安纳步行者队的比赛中，麦格雷迪代表猛龙上场23分钟，得到10分和11个篮板，收获职业生涯第一次两双，并以18岁174天的年龄创造了NBA历史上自1985-86赛季有单场个篮板统计以来，得两双的最年轻球员纪录 [7] 。

1998年2月13日，在猛龙115-130负于布鲁克林篮网队的比赛中，麦格雷迪代表猛龙替补出战37分钟，全场16投7中、罚球10罚8中，砍下了全队第二高的22分和8个篮板，并创造了1997届新秀单场最高得分纪录。

1997-98赛季，麦格雷迪代表猛龙出场64场比赛，场均出场18.4分钟，得到7分、4.2个篮板、1.5次助攻和1封盖，投篮命中率达到45%，创造了NBA历史上21岁以下球员单赛季最高投篮命中率的记录 [8] 。

1998-99赛季，麦格雷迪因受背伤困扰，代表猛龙出场49场比赛，场均出场22.6分钟，得到9.3分、5.7个篮板和2.3次助攻 [9] 。

2000年3月19日，在猛龙100-98战胜休斯敦火箭队的比赛中，麦格雷迪代表猛龙出场43分钟，全场16投8中得到16分、7个篮板、4次助攻、2次抢断和7次封盖，刷新个人职业生涯单场封盖纪录。

1999-00赛季，麦格雷迪代表猛龙出场79场比赛，场均出场时间31.2分钟，得到15.4分、6.3个篮板和3.3次助攻 [6] ；该赛季，麦格雷迪得到了生涯单季最高的151次封盖，成为加·赫德、特里·泰勒、波·奥特洛、埃尔顿·布兰德以及丹·罗德菲尔德之后的赛季封盖数量第六人 [7] 。

魔术时期

2000年休赛期，麦格雷迪以7年9300万美元的合同加入奥兰多魔术队，选择了昔日偶像哈达威的1号球衣。

2000年12月2日，在魔术95-74战胜篮网的比赛中，麦格雷迪代表魔术出场42分钟，全场得到40分、10个篮板、5次助攻、3次抢断和3个封盖，刷新个人职业生涯单场得分纪录，并成为NBA历史上自乔·巴里·卡罗尔、哈基姆·奥拉朱旺、大卫·罗宾逊以及拉里·伯德之后第五位得到如此数据的球员 [10] 。

2001年4月13日，在魔术110-113负于华盛顿奇才队的比赛中，麦格雷迪代表魔术出场44分钟，全场18投10中，得到49分、5个篮板和4次助攻，创个人职业生涯单场得分新高。

2000-01赛季，麦格雷迪有连续9场25+得分、77场比赛76场得分上双、26次30+得分的表现，代表魔术出场77场比赛，场均出场40.1分钟，得到26.8分、7.5个篮板和4.6次助攻，场均得分排名联盟第7，同时也成为继沙奎尔·奥尼尔、迈克尔·乔丹后又一位在21岁之前就能得到场均26分以上的球员 [10] ；赛季结束后，麦格雷迪获得NBA进步最快球员奖。此外，麦格雷迪还入选了年度最佳阵容二阵和全明星阵容 [11] 。

2002年2月23日，在魔术105-87战胜费城76人队的比赛中，麦格雷迪代表魔术出场35分钟，全场得到22分、11个篮板和11次助攻，并在个人职业生涯中首次获得“三双”数据。

2002年3月8日，在魔术99-96战胜奇才的比赛中，麦格雷迪代表魔术出场42分钟，全场得到50分、10个篮板和3次助攻，成为NBA历史上年龄第三小获得“50+10”数据的球员，仅次于贾马尔·马什本和沙奎尔·奥尼尔，同时刷新个人职业生涯单场得分纪录。

2001-02赛季，麦格雷迪代表魔术出场76场比赛，场均出场38.3分钟，得到25.6分、7.9个篮板和5.3次助攻 [10] 。

2002年12月1日，在魔术117-92战胜亚特兰大老鹰队的比赛中，麦格雷迪代表魔术上场30分钟，全场19投12中得到28分、3个篮板和6次助攻，刷新个人职业生涯单场命中率记录。

2002年12月26日，在魔术104-99击败底特律活塞队的比赛中，麦格雷迪代表魔术上场44分钟，全场26投14中，得到46分和6个篮板，并刷新个人职业生涯罚球命中数记录 [12] 。

2003年2月22日，在魔术110-96战胜芝加哥公牛队的比赛中，麦格雷迪代表魔术上场33分钟，全场32投15中得到52分、5个篮板和3次助攻，同时刷新个人职业生涯单场得分记录。

2003年2月23日，在魔术113-105战胜篮网的比赛中，麦格雷迪代表魔术上场45分钟，全场得到46分、10个篮板和13次助攻的“三双”数据，成为NBA历史上既拉里·伯德、迈克尔·乔丹、米歇尔·亚当斯和克里斯·韦伯之后第四位得到“40+10+10”数据的球员。

2003年3月6日，在魔术111-99战胜密尔沃基雄鹿队的比赛中，麦格雷迪代表魔术上场43分钟，全场34投17中得到48分、10个篮板和5次助攻，刷新个人职业生涯运动战进球数记录。

2003年3月24日，在魔术96-74战胜热火的比赛中，麦格雷迪代表魔术上场32分钟，全场27投15中砍下37分、7个篮板和5次助攻，其中三分球11投7中，刷新个人职业生涯单场三分球命中数记录 [12] 。

2002-03赛季，麦格雷迪代表魔术出场75场比赛，场均出场39.4分钟，得到32.1分、6.5个篮板和5.5次助攻，成为1980年后，既迈克尔·乔丹、伯纳德·金、乔治·格文之后第四位单赛季场均得分高于32.1分的球员，同时获得该赛季NBA得分王 [10] 。

2003年11月15日，在魔术101-106负于丹佛掘金队的比赛中，麦格雷迪代表魔术上场41分钟，全场30投20中砍下51分、6个篮板和5次助攻，其中三分球11中8，总命中率66.7%，刷新个人职业生涯单场投篮命中率记录和个人职业生涯单场三分命中数记录 [13] 。

2004年3月11日，在魔术108-99战胜奇才的比赛中，麦格雷迪代表魔术出场46分钟，全场37投20中，罚球26罚17中得到62分、10个篮板和5次助攻，刷新个人职业生涯单场出手数、得分、运动战进球数和罚球命中数记录。随后，麦格雷迪因伤结束了赛季。

2003-04赛季，麦格雷迪代表魔术出场19场比赛，场均出场39.9分钟，得到24.8分、6.4个篮板和5.3次助攻，并蝉联NBA得分王 [10] [14] 。

火箭时期

2004年夏天，麦格雷迪离开魔术，休斯顿火箭队用史蒂夫·弗朗西斯、卡蒂诺·莫布里和凯文·卡托为筹码交易来了麦格雷迪、朱万·霍华德、泰伦·卢以及利斯·盖因斯 [15] 。

2004年12月9日，在火箭81-80战胜圣安东尼奥马刺队的比赛中，麦格雷迪在比赛最后的35秒中投中3个三分球外加一个三分加罚得到13分；共出场40分钟，全场得到33分、8个篮板和2次助攻，率领火箭队以81-80逆转马刺队，这一表现也被称为“麦迪时刻” [16] 。

2004-05赛季，麦格雷迪代表火箭出场78场比赛，场均出场40.8分钟，得到25.7分、6.2个篮板、5.7次助攻、1.7次抢断以及0.7个封盖，并成为火箭队史上第一个能够同时达到单赛季2000分、400个篮板和400次助攻数据的球员；这个赛季，麦格雷迪被选为全明星赛西部首发球员 [17] ，麦格雷迪在赛季结束后还入选了NBA最佳阵容三队 [18] 。

2005-06赛季，球队遭到伤病困扰，麦格雷迪赛季只代表火箭出场47场比赛，场均出场37.1分钟，得到24.4分、6.5个篮板和4.8次助攻 [6] 。

2006-07赛季，麦格雷迪代表火箭出场71场比赛，场均出场35.8分钟，得到24.6分、5.3个篮板和6.5次助攻，刷新个人职业生涯场均次助攻数记录。而麦格雷迪也入选了全明星赛二阵 [10] 。

2007-08赛季，麦格雷迪代表火箭出场66场比赛，场均出场47分钟，得到21.6分、5.1个篮板和5.9次助攻 [19] 。

2008-09赛季，麦格雷迪因伤只代表火箭出场35场比赛，场均出场33.7分钟，得到15.6分、4.4个篮板和5次助攻，命中率只有38.8% [10] 。

宣布退役

2013年8月26日，麦格雷迪正式宣布退役，结束16年的职业生涯。麦格雷迪职业生涯中共出场938次，平均出场时间32.7分钟，场均5.6个篮板，4.4次助攻和1.2次抢断，场均得分19.6分 [1] 。

2016年12月，麦格雷迪被提名成为2017年奈·史密斯篮球名人纪念堂的候选人 [30] 。

2017年4月2日，麦格雷迪正式入选2017年奈·史密斯篮球名人纪念堂 [2] 。

赛场之外

个人生活

婚姻情况

高中时，麦格雷迪认识了妻子克莱伦达·哈里斯。2003年1月，他们的女儿莱拉来到人世，2006年，二人迎来了第二个孩子雷蒙·拉玛尔。同年二人正式宣布结婚 [31] 。

兄弟关系

麦格雷迪是NBA球星文斯·卡特的远亲表弟 [30] 。

绰号

麦格雷迪绰号“麦迪”/“麦蒂”或“T-Mac”。

社会生活

2008年夏天，麦格雷迪前往非洲贫穷地区，探望当地的难民，并当 选美国年度慈善名人 [32] 。

2009年访华期间，麦格雷迪被红十字基金会聘为“温暖中国大使”，并参和了当年的温暖中国行活动 [32] 。

2011年，麦格雷迪经中国红十字基金会邀请，作为形象大使应邀来到中国发展慈善事业；1月，麦格雷迪自掏腰包捐款2万美元，用以帮助一位在运动中受伤的橄榄球运动员 [32] 。

2016年2月，麦格雷迪受邀参加多伦多全明星名人赛，全场得到16分、6个篮板、4次助攻和2次抢断，并担任全明星扣篮大赛评委 [33] 。

2016年4月1日，麦格雷迪客串解说雷霆主场119-117击败快船的比赛 [34] 。

2016年4月24日，在励志竞技体育节目《来吧冠军第一季》第四期中，麦格雷迪和易建联组成篮球“冠军队”，迎接“终结者队”的挑战 [35] 。

社会评价

麦格雷迪是我所见过的所有篮球运动员中天赋最高的，甚至连迈克尔·乔丹与科比·布莱恩特都要往边上靠靠 [40] 。（前NBA球星格兰特·希尔评）

麦格雷迪是火箭的“第四节先生”，在姚明全力支撑住前三节后，麦格雷迪就是决定火箭胜负的“关键先生” [41] 。（美国篮球专家丹尼·里维斯评）

麦格雷迪是一个拥有辉煌过去的明星球员。他两度荣膺联盟得分王，也有过属于自己的35秒13分神迹。可以说，正因为他的惊艳表现，才让人们对他有着这么高的期待，继而由于落差而感到同样巨大的落差。作为以前和科比相提并论的天才球员，在很多人眼中，麦格雷迪的天赋甚至可以与“篮球之神”乔丹媲美 [42] 。总结麦格雷迪的生涯，从球队荣誉方面去看，他并不成功，但就个人而言，他足够光彩 [6] 。（网易评）

两届得分王麦格雷迪曾经被认为是天赋超过科比的球员。他创造的35秒13分神迹至今未球迷们津津乐道。然而，随着伤病的来袭，原本那个具备顶尖水准的球员逐渐疲软，场上场下都变得慵懒，总是找各种理由缺席比赛 [43] 。（腾讯网评）

在麦格雷迪的职业生涯中，他每次复出时都会有惊人的表现，为此他赢得了一个“复出先生”的美名 [44] 。麦格雷迪是火箭在球场上的主心骨 [45] 。（新浪网评）
